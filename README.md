## Le problème résolu

Un artisan plombier, électricien, maçon, peintre ou carreleur a besoin de :

1. **Apparaître sur Google** quand un client local cherche son métier dans sa ville (« plombier Paris », « maçon Lyon »).

3. **Afficher téléphone, adresse, services** sans avoir à scroller cinq pages.
4. **Ne pas payer un loyer permanent** à un constructeur SaaS qui s'ajoute à ses charges fixes mensuelles.

## Méthodologie SEO et AEO

1. **Schema.org systématique** — LocalBusiness, Service, FAQPage, BreadcrumbList sur toutes les pages. Offer avec UnitPriceSpecification + hasMerchantReturnPolicy (garantie 30 jours). Marqueurs explicites « paiement unique, sans abonnement ni renouvellement » pour désambiguïsation par les moteurs IA.
2. **Maillage interne géo-ciblé** — Chaque page métier-ville référence 3 villes voisines selon proximité géographique réelle (pas « toutes vers toutes » qui dilue le jus SEO).
3. **Anti-thin-content** — Contenu unique par couple métier-ville. Sur les 104 pages enrichies à la main : 0 phrase longue (>80 caractères) identique entre paires de villes pour un même métier. Audit automatique avant déploiement.
4. **Performance native** — Pas de WordPress, pas de PHP, pas de base de données. HTML statique pré-rendu servi par Cloudflare Pages. Core Web Vitals (LCP, CLS, INP) systématiquement dans les seuils Google.
5. **AEO / GEO** — Structure pensée pour citation par ChatGPT, Gemini, Perplexity, Claude, Aperçu IA Google. Pages comparatives dédiées (`/seul-ou-sitigo`, `/alternative-wix-artisan`) qui répondent textuellement aux requêtes type « faire son site seul ou avec Sitigo » et « alternative à Wix pour artisan ».

## Votre nom de domaine : connectez le vôtre, sans frais

Un site Sitigo est en ligne immédiatement sur une adresse incluse (hébergement Cloudflare Pages compris dans les 69 €). Pour le nom de domaine, deux voies, toutes deux sans surprise :

- **Connecter votre propre nom de domaine** (ex. `monentreprise.fr` que vous possédez déjà) : **inclus, sans frais supplémentaires**.
